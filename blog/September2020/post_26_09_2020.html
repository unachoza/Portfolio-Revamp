Counting Sort

Not Like The Others

26Sep 2020 Arianna Choza

Counting Sort, not a comparison sort. Just blew my mind. The previous 5 sorting algorithms I learned all take unsorted integer arrays inputs and magically spit them out as sorted integer arrays. How? By comparing numbers, finding the greater numbers toward the end of the array based on the previous number it is being compared to. How else do you find the greater numbers, if not by comparing them to smaller numbers to determine the relative greatness??

Let me introduce Counting Sort.The number of occurrences of an integer are kept track of, then a prefix array is made, the numbers are put into a new array using the count array that was just made. Absolutly Nuts!!
